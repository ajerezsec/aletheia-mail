# Roadmap

## MVP Milestones

1. Project foundation and tooling baseline.
2. Domain models and typed contracts.
3. Robust `.eml` parser.
4. Independent analyzers:
   - Header Analyzer
   - URL Analyzer
   - Attachment Analyzer
5. Findings engine and risk scoring.
6. Report generation (Rich, JSON, Markdown).
7. End-to-end CLI orchestration.
8. Test expansion and documentation hardening.

## Post-MVP Extensions

- VirusTotal integration
- URLScan integration
- AbuseIPDB integration
- AlienVault OTX integration
- Shodan enrichment
- HTML report export
- REST API with FastAPI
- Docker support
- GitHub Actions CI/CD
- Plugin system
- Custom rules engine
- YARA-based attachment analysis
- PDF report generation
